# Rich Menu 圖片素材

請將以下三張圖片放置在此目錄：

## 1. taiwan-stock.jpg
- **用途**：台股分析按鈕背景
- **建議尺寸**：833 x 843 px（或更大）
- **內容**：股市看板圖片（綠色數字顯示）

## 2. us-stock.jpg
- **用途**：美股分析按鈕背景
- **建議尺寸**：833 x 843 px（或更大）
- **內容**：川普圖片或美股相關圖片

## 3. rating.jpg
- **用途**：本週評分按鈕背景
- **建議尺寸**：833 x 843 px（或更大）
- **內容**：金色數字5圖片

## 圖片要求

- **格式**：JPG 或 PNG
- **尺寸**：建議至少 833 x 843 px
- **品質**：高清圖片，避免模糊
- **色調**：
  - 台股：綠色調
  - 美股：藍色調
  - 評分：金色/橙色調

## 使用方式

1. 將三張圖片放置在此目錄
2. 執行：`npm run generate:richmenu:images`
3. 生成的 Rich Menu 圖片會保存在 `public/rich-menu.png`
